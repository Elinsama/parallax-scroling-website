Distributions

Ubuntu

Ubuntu is the most widely used Linux distribution. The distribution is sponsored by the company Canonical. The philosophy of Ubuntu is to provide an easy-to-use OS that also appeals to beginners. Ubuntu is based on the Debian distribution.
The Ubuntu versions are named after the year and the month (year.month) and get a special project name like Focal Fosa or Hirsute Hippo. Versions are released in April and October. The version in April is the LTS (long term support) version which received updates for four years which is why it is recommended to use this version opposed to the version in October.

Debian

Debian is a free , collaboratively developed OS. Debian GNU/Linux is based on the basic system tools of the GNU project and the Linux kernel. In addition, Debian has a large selection of packages. Currently, there are over 57.000 program packages. Debian uses dpkg and apt as the frontend of the package management.

Fedora

Fedora is a Linux distribution sponsored by Red Hat. The operations system consists only of free software. The aim of the projects is to make this available to a target group.

openSUSE

“The makers choice for system administrator, developer and desktop users”
OpenSUSE is a community project that promotes the use of Linux and free software. In addition to several Linux distributions, tools for development and testing are developed. These tools include OBS, openQA, YaST and Kiwi.

There are two versions of openSUSE:

  • Tumbleweed (rolling releases) and
  • Leap.

Linux Mint

The main edition of Linux Mint is a distribution based on Ubuntu. There variants of desktops environments are available:

  • Cinnamon,
  • MATE and
  • Xfce

Cinnamon is the desktop environment specially developed by Mint. This is intended to make the integration of the distribution simple and user-friendly.

Arch

Arch is a linux distribution with rolling releases. This means that the OS will be updated constantly. The development team follows the KISS principle. The OS is designed for advanced users. Arch uses pacman (packagemanager) for package management. It is comparable with apt. It can resolve dependencies automatically and download the required packages from the Arch repositories. If you have a problem with Arch, it is recommended to visit the Arch wiki.

Gentoo

Gentoo is a source-based Linux distribution. It is for very advanced users only! To install this OS, you have to choose the packages you want to install and the user has to compile the whole thing. “There is no installation, you are the installer”. That way, you can apply all the customization you desire. This requires experience with Linux in general.
