Commands

Why use the command line?

To use Linux most efficiently, you should use the command line. Anyone who masters it can work more efficiently and gain a lot of control over the system. Here we give you a brief overview. If you need further help visit Stackoverflow or Ubuntuusers.

Stackoverflow
Ubuntuusers

ls

ls is probably the first command every Linux user types in their terminal. It prints content of the directory you want to show including files and directories.

cd

The cd command is very popular, along with ls. It stands for “change directory” and, as its name suggests, switches you to the directory you’re trying to access. If you want to access Videos you should type “cd Videos/”.

cp

It’s so easy to copy files and folders directly in the Linux terminal that sometimes it can replace conventional file managers.

rm

Now that you know how to copy files, it`ll be helpful to know how to remove them. Rm is used to remove files out of directories. Be careful while using it, though, because it’s very difficult (yet not impossible) to recover files deleted this way.

mv

You can use the mv command to move (or rename) files and directories through your file system.

mkdir

To create folders in the terminal, you will need to use the mkdir command. Just specify the new folder`s name, ensure it doesn’t exist, and you’re ready to go.

chmod

The chmod lets you change the mode of a file (permissions) quickly. It has a lot of options available with it.

apt/yum/dnf/pacman

No matter which Linux distribution you’re using, it’s likely you use package managers to install, update, and remove the software you use every day.
You can access these by the terminal through the command line, and you’d use one or another depending on the distro you are running.

cat

It is short for “concatenate”, which lets you create a view, and concatenate files directly from the terminal. It’s mainly used to preview a file without opening a graphical text editor like nano or vim.

ping

Ping is a simple command, It’s used for Networking; ping has a ton of options but the most common cases are that you use an IP-Address or a Domain-Name to “ping” it.

less

Less is the opposite of more. It’s a program that lets you inspect files backwards and forwards.

more

More is the opposite of less, it’s a program that lets you read files.

pwd

The pwd command stands for “print working directory”, and it outputs the absolute path of the directory you’re in . For example if your username is “user” and you’re in your Documents directory, its absolute path would be: /home/user/Documents.

touch

With touch you can create a file.

sudo

This command stands for “superuser do” and it lets you act as a superuser or root user while you’re running a specific command. It’s how linux protects itself and prevents users from accidentally modifying the machine’s filesystem or installing inappropriate packages.

It is commonly used to install software or to edit files outside the user’s home directory.

htop

htop is an interactive process viewer that lets you manage your machine’s resources directly from the terminal. In most cases, it isn’t installed by default, so make sure to read more about it on its download page.

unzip

The unzip command allows you to extract the content of a .zip file from the terminal. Once again, this package may not be installed by default, so install it with your package manager.

zip

The zip command allows you to zip a folder into a .zip file.

echo

The echo command is used to display lines of text/string that are passed as an argument. this is a built in command that is mostly used in shell scripts batch files to output a status text on screen.

kill

It’s annoying when a program is unresponsive, and you can’t close it by any means. Fortunately, the kill command solves this kind of problem. it will “kill” any process without hesitation.

wget

wget (world wide web get) is a utility to download content from the internet. It has one of the largest collections of flags out there.

man

Another essential Linux command is man. it displays the manual page of any other command (as long as it has one).

shutdown/reboot

As you may guess, the shutdown command lets you power off your machine. It also can be used as a reboot command. For this command you need Root permissions!

whoami

The whoami command displays the username with which you are currently logged in.

uname

uname (short for “unix name”) prints the operative system information, which comes in handy when you know your current linux version.

history

when you are struggling to remember a command you have used before you may use history to see the list of commands you have used in the current terminal.

passwd

passwd allows you to change the passwords of user accounts. First, it prompts you to enter your current password, then asks you for a new password and confirmation.

head

This one is complementary to the tail command. head outputs the first 10 lines of a text file, but you can set any number of lines you want to display with the -n flag.

tail

tail is as said with the command head it displays the lines of the text, this time just the last lines instead of the first.

grep

grep is one of the most powerful utilities for working with text files. it searches for lines that match a regular expression and then it prints them.

find

The find command searches for files in a directory hierarchy based on a regex expression.

hostname

A computer in the network can be identified by its “hostname”. This command lets you change it.

useradd

useradd is a command that lets you add a user to your system.

userdel

userdel is the opposite of useradd; it is used to remove/delete a user.

systemctl

systemctl is used to utilize and control/manage systemd and services/demon’s. You can see it as a control interface to manage your system’s services.

clear

clear is a command that clears your terminal for you.

date

The date command is used to display the system date and time.

pstree

pstree (process tree) is a linux command that shows the running processes as a tree. It is used as a more visual alternative to the ps command.

chown

The chown command lets you change the ownership of a given file, directory, or symbolic link. Because in linux all files are associated with an owner.

diff

diff is a command that lets you compare two files line by line. So if you’d need to compare them. This lets you do that easily.

tar

The tar program stands for tape archiver. It was originally written to back up data on tape drives. Even today it is the most popular tool for archiving data on Linux systems.

du

du stands for disk usage. It displays the disk usage of the system.

df

df short for disk free, is used to display information about how much disk space is in use and how much is free.

locate

The command locate is used to locate the file you are looking for on your system.

neofetch

neofetch is a command that is not standart, but it shows you a nice graph on what kind of system you are running. Mostly used it’s used to flex your system to other linux users.

cli-pride-flags

This command lets you display a pride flag.
