The Linux Kernel

What is the Linux Kernel?

The Linux kernel is the main component of the Linux operating system. It is the central interface between a computer’s hardware and its processes.

Functions are:

  1. Memory management: Tracks how much memory is used to store data and where it is stored.
  2. Process management: Determines which processes can use the CPU (Central Processing Unit), when and for how long.
  3. Device driver: Serves a medicator or translator between hardware and processes.
  4. System calls and security: Receives service requests from the processes.

The structure of a kernel can be visualized using the onion skin model.
